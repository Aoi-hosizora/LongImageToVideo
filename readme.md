# Long Image To Video

### Environment
+ `Python` 3.6
+ `numpy` 
+ `PIL.Image`
+ `OpenCV ` 3.3.0

### Run
```python
# python longimg.py 
#	<Img_folder> <Extension> <Resolution: W H> <FPS> <PixelMove> <Frame: Begining Waiting> <Save filename>

python longimg.py "xxxxxx\ED" png 16 9 60 1 50 100 final
```
